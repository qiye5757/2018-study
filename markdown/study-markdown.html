标题

  • 使用 = 或者 - 来表示一级标题和二级标题
  • 使用 # 可以表示6级标题

段落

  • 段落的前后要有空行（在段落中实现强制换行需要两个空格+一个换行）

区块引用

  • 使用 > ,可以嵌套

代码区块

  • 使用两次 四个空行或者一个tab

强调

  • ** 或者 __ 为粗体， * 或者 _ 为斜体

列表

  • 使用 · 或者 + 或者 - 表示无序列表

分割线

  • 使用 三个以上的 * 或者 - 或者 _

链接

  • 行列式或者参考式

行列式如：[a](aaaaaa)
显示 a

参考式如：[122222][222]
[222]:aaaaaa
显示1222222

图片

  • 图片和链接相似，只是在前面加一个！

反斜杠

  • 相当于转义的作用

符号`aaaa`

  • 起到标记的效果
